# RFID 125
+ Kirova.rfid - Работает на многих стандартных домофонах. В основном на улице Кирова.
+ VezdeXod.rfid - Данный ключ подходит к Очень многим стандатным домофонам (поэтому он так и называется (・∀・))

# NFC
+ Smart_intercom - Этот ключ подходит к ограниченному количеству домофонов "Умный домофон", но он очень редко подходит... (￢_￢)
+ Solnechnya - Этот же ключ подходит к обычным NFC домофонам, в особенности на Солнечном проезде.
